# Description
Easy Python console tool wrappers

## Install

```
pip install xy_type
```

## Start

```python
from xy_type.utils import count_of_function_arguments, validate_callable_func, validate_callable_init_func, has_func, empty_value

class xyObject:
    def xyFunc(arg_0=0, arg_1=1):
        pass
```
